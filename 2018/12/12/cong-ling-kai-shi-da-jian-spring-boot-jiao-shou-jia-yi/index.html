java 脚手架
spring boot 微服务 2018-12-12

spring boot相关概念先请各位大佬自行百度下，有时间可以发个博客介绍下。

项目构建

作者用idea开发，idea初始化spring boot项目，一般是最新稳定版。
构建过程如下：

file->new->project->spring initialize->jdk版本->maven相关配置->选取依赖（首先选web,其它可以以后添加）

初始化后pom.xml：

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>
    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>2.1.1.RELEASE</version>
        <relativePath/> <!-- lookup parent from repository -->
    </parent>
    <groupId>com.siteqi.cn</groupId>
    <artifactId>scaffold</artifactId>
    <version>0.0.1-SNAPSHOT</version>
    <name>scaffold</name>
    <description>Demo project for Spring Boot</description>

    <properties>
        <java.version>1.8</java.version>
    </properties>

    <dependencies>
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>

        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-test</artifactId>
            <scope>test</scope>
        </dependency>
    </dependencies>

    <build>
        <plugins>
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
            </plugin>
        </plugins>
    </build>

</project>

初始化后一般生成如下目录：

  • main 主要放置项目启动类：项目名+Application eg：ScaffoldApplication。
  • test 主要放置项目启动测试类类。
  • resource 静态资源 最重要的是application.properties，你可以在这里配置所有配置，也可以用yml格式的配置文件。

配置测试

  1. 建立一个controller包，创建以下测试类：
/**
 * 测试Controller
 *
 * @author wangmaoyu
 * @create 2018-12-12 22:38
 **/
@RestController
@RequestMapping("/test")
public class TestController {
    @GetMapping("/hello")
    public String hello(){
        return "这是我第一个spring boot项目！";
    }
}
  1. application.yml配置如下：
server:
  port: 8080
  1. 这样运行ScaffoldApplication启动类，启动成功后即可在浏览器中http://localhost:8080/test/hello访问接口，浏览器里显示：
这是我第一个spring boot项目！
